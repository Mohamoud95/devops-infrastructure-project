# Screenshots

Place sanitized implementation screenshots in this folder.

Recommended naming format:

- `phase-1-1-ufw-status.png`
- `phase-1-1-fail2ban-status.png`
- `phase-1-1-auditd-rules.png`
- `phase-1-2-pki-directories.png`
- `phase-1-2-ca-certificate.png`

Before uploading screenshots publicly, check that they do not reveal secrets, tokens, passwords, private keys, or sensitive cloud credentials.
